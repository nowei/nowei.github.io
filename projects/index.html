ChatAssist

Context
CSE 481V - VR Capstone
Description
AR app using MagicLeap that would transcribe/translate English to other languages in AR to help people who are hard of hearing or are still learning English. Basically captions in real life. We had a setup of bluetooth microphones that streamed English conversations to laptops that were then forwarded to Amazon Transcribe. We then updated the text displayed on the Magic Leap to whatever language was selected. We originally used object detection for markers to mark where the text boxes would go, but learned that the Magic Leap's image tracking wasn't super stable, so we ended up making it adjustable instead.
Technologies
C#, Unity
Contribution
Designed the controls for changing languages and scrolling through the chat log, experimented with the computer vision detection on the Magic Leap to determine where to place the markers for the chat bubble. Dealt with random Unity bugs.

Relief

Context
MITHacks 2018
Description
Disaster relief app for reporting missing people. Developed a java application to help disaster relief volunteers coordinate between disaster relief camps and people outside disaster areas to look up information on friends and relatives. Used Microsoft Azure to run a database for keeping track of found and missing people at disaster areas and Microsoft Cognitive Services Face API to look for people using images. Allowed friends and family to upload information/photos to the application to assist relief efforts, like where people could potentially be and what they looked like so that they could identify them using computer vision. Then the volunteers could update the information to include which relief camp people are located in.
Technologies
SQLServer, Java, Azure
Contribution
Led project, set up the Microsoft Azure database and backend that would interpret data sent from the front end, interact with the server, and send the information back to the front end. Helped determine what kind of information we would store.
